# Web Toys 1000 - 完整開發計畫

> 1000 個純前端互動 Web Toys 的完整規劃文件

## 計畫總覽

本計畫將 1000 個 Web Toys 分為 10 大類別，每個類別包含 100 個 toys。每個分類有獨立的詳細規劃文件。

## 統一規格

### 每個 Toy 包含

```
toys/XXX-toy-name/
├── index.html      # 入口頁面
├── style.css       # 樣式
├── script.js       # 主程式
├── preview.png     # 預覽圖 (800x600)
├── preview.gif     # 動態預覽 (可選)
└── README.md       # 說明文件
```

### 表格欄位說明

| 欄位 | 說明 |
|-----|------|
| 編號 | 001-1000 的唯一識別碼 |
| 名稱 | 中英文名稱 |
| 玩法說明 | 使用者如何互動、可以做什麼 |
| 技術重點 | 使用的核心技術與 API |
| 瀏覽器需求 | 最低瀏覽器版本要求 |
| 進度 | 待開發 / 開發中 / 已完成 / 已測試 |

### 瀏覽器需求等級

| 等級 | 說明 | 適用 Toys |
|-----|------|----------|
| Level 1 | 所有現代瀏覽器 (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+) | ~600 |
| Level 2 | 需要較新瀏覽器 (Chrome 90+, Firefox 90+, Safari 15+) | ~300 |
| Level 3 | 需要最新瀏覽器 + 實驗性功能 (WebGPU, etc.) | ~100 |
