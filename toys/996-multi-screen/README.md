# Multi-Screen Window Manager

Demonstrates the **Window Management API** (formerly Multi-Screen Window Placement).

## Features
- Detects all connected monitors/screens.
- Shows resolution, position, and labels of screens.
- Allows spawning a popup window specifically positioned on a chosen screen.

## Requirements
- Desktop Browser (Chrome / Edge 100+)
- Permission must be granted to manage windows.
- Note: Some browsers may require enabling a flag or specific permission settings.
